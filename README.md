#  Project : E-SEKAI !!
## 팀명 : 좌냥 하 우멍

<br />

## 📌 팀원 소개 및 담당
- 안주영 : 컨트랙트 및 페이지 기능
- 김종찬 : 전체 페이지 UI 및 페이지 관리

<br />

* 목표
  * 우리는 그동안 배워온 지식을 바탕으로 블록체인과 지갑을 연결한 NFT 거래소를 구현하는 것을 목표한다.
  * 실제 거래경험을 바탕으로 유저에게 편리한 UI를 제공, 직접 토큰을 발행함으로써 토큰의 유통과정 뿐만 아니라 생성원리에 대해서 공부한다.
  * 더 나아가 직접 개발한 토큰을 거래소에서 사용할 수 있게 만드는 것을 목표한다.
